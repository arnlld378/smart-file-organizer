# 📁 Smart File Organizer
Simple and efficient Python script to keep your folders clean!

## 🚀 Features
* **Auto-sorting:** Groups files into Images, Documents, Videos, and more.
* **Logging:** Keeps track of every moved file in `activity_log.txt`.
* **Safe:** Handles errors if files are busy or restricted.

## 🛠️ Installation & Usage
1. Install Python 3.x
2. Download `organizer.py`
3. Run the script:
   ```bash
   python organizer.py
